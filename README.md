## 程序的功能
1. 浏览英雄属性
2. 浏览物品属性
3. 计算英雄加上物品后的属性

## Activity
+ MainActivity
  + 搜索框
  + 推荐英雄
  + 推荐物品
+ CalculatorActivity
+ HeroActivity
  + 显示英雄信息
    + 图片
    + 名称
    + 技能
    + 推荐出装
+ ItemActivity
  + 显示物品信息
    + 图片
    + 名称
    + 属性
+ AboutActivity
  + 显示帮助文档
  + 显示开发人员信息
  + 显示程序版本
